# 美国斩杀线生存指南

## 什么是"斩杀线"？

"斩杀线"指的是当美国普通人（尤其是留学生）的**财务储备、精神支撑与生存资源**跌破临界点后，会触发不可逆的连锁崩盘，最终被社会系统"淘汰"的生存阈值。

一旦越过斩杀线，你可能面临：
- 医疗账单摧毁信用记录
- 无力支付房租被驱逐
- 失去合法身份（F-1/OPT/H-1B断签）
- 精神崩溃无法继续学业或工作
- 陷入债务螺旋无法翻身

**本指南的目标**：帮助在美中国留学生和华人**提前识别风险、掌握预防策略、学会维权技巧**，避免跌入斩杀线。

---

## 为什么需要这个指南？

在美国生活，你会发现：
- **大型语言模型（GPT/Claude等）不愿意告诉你的实战技巧**
- 医疗账单可以砍价，而且可以砍很多（30-80%减免）
- 信用卡争议（dispute）是合法且有效的维权手段
- F-1学生被排除在几乎所有联邦福利之外（SNAP, Medicaid, 失业救济全部不eligible）
- 失去住房会触发20+连锁崩溃效应，几乎不可逆
- 知道规则和不知道规则的人，命运完全不同

**这个仓库收集的是那些"教科书不教、AI不敢说、但你必须知道"的生存知识。**
