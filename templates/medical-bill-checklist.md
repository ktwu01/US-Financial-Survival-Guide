# 医疗账单检查清单

> 使用此清单逐项检查你的医疗账单,发现错误可以节省数千美元!

## 基本信息核对

- [ ] **患者信息**
  - [ ] 姓名拼写正确
  - [ ] 出生日期正确
  - [ ] 地址正确
  - [ ] 联系方式正确

- [ ] **保险信息**
  - [ ] 保险公司名称正确
  - [ ] 保单号码正确
  - [ ] 保险已经处理(Claim Processed)
  - [ ] 保险支付金额正确显示

- [ ] **就诊信息**
  - [ ] 就诊日期正确
  - [ ] 就诊地点正确
  - [ ] 医生姓名正确
  - [ ] 诊断代码(ICD codes)正确

---

## 收费项目检查

### 服务项目核对
- [ ] **所有服务都是我实际接受的**
  - [ ] 没有陌生的服务项目
  - [ ] 没有我拒绝的服务
  - [ ] 没有取消的预约或服务

- [ ] **数量正确**
  - [ ] 药品数量与实际相符
  - [ ] 检查次数正确
  - [ ] 治疗次数正确

- [ ] **日期匹配**
  - [ ] 所有服务日期与我的就诊记录一致
  - [ ] 没有我不在医院的日期

### 常见错误检查

- [ ] **重复收费**
  - [ ] 同一项服务只出现一次
  - [ ] 同一天的相同检查没有重复
  - [ ] 药品没有重复收费

- [ ] **编码错误**
  - [ ] CPT代码(医疗程序代码)与实际服务匹配
  - [ ] ICD代码(诊断代码)与诊断结果一致
  - [ ] 没有明显的编码错误(如用错误代码导致更高收费)

- [ ] **不合理收费**
  - [ ] 价格与该地区平均水平相符(可查询 Healthcare Bluebook)
  - [ ] 没有异常高的"设施费"(Facility Fee)
  - [ ] 没有不合理的"急诊费"(Emergency Room Fee)

---

## 保险处理检查

- [ ] **保险公司已处理**
  - [ ] EOB(Explanation of Benefits)已收到
  - [ ] EOB上的金额与账单一致

- [ ] **保险覆盖正确**
  - [ ] 应该覆盖的服务被正确覆盖
  - [ ] Deductible(自付额)计算正确
  - [ ] Co-pay(定额共付)金额正确
  - [ ] Co-insurance(共同保险)比例正确
  - [ ] Out-of-pocket maximum(自付上限)计算正确

- [ ] **网络内/外正确分类**
  - [ ] 网络内医生按网络内费率收费
  - [ ] 没有意外的"out-of-network"收费

---

## 折扣与减免检查

- [ ] **可申请的折扣**
  - [ ] 询问"uninsured discount"(无保险折扣)
  - [ ] 询问"prompt pay discount"(快速支付折扣)
  - [ ] 询问"self-pay discount"(自费折扣)
  - [ ] 询问"financial hardship discount"(经济困难折扣)

- [ ] **慈善医疗减免**
  - [ ] 医院是否为非营利机构
  - [ ] 是否有Charity Care Program
  - [ ] 我是否符合申请条件
  - [ ] 已获取申请表格

---

## 付款信息核对

- [ ] **付款期限**
  - [ ] 了解最晚付款日期
  - [ ] 了解争议时限
  - [ ] 了解慈善医疗减免申请时限

- [ ] **付款选项**
  - [ ] 询问分期付款计划
  - [ ] 询问无息分期
  - [ ] 了解最低月付金额

---

## 发现错误后的行动清单

如果发现错误,立即采取以下行动:

1. **记录错误**
   - [ ] 拍照保存账单
   - [ ] 列出所有发现的错误
   - [ ] 准备证据(就诊记录、收据等)

2. **联系医院/诊所**
   - [ ] 拨打账单上的客服电话
   - [ ] 要求与Billing Department(账单部门)对话
   - [ ] 记录对话时间、对方姓名、工号
   - [ ] 要求书面确认错误更正

3. **书面争议**
   - [ ] 使用 [医疗账单争议信模板](medical-bill-dispute-letter.md)
   - [ ] 附上证据副本
   - [ ] 通过挂号信寄送(Certified Mail)
   - [ ] 保留所有副本

4. **跟进**
   - [ ] 14天后电话跟进
   - [ ] 30天内要求书面回复
   - [ ] 如未解决,升级投诉

## 有用工具和资源

### 价格查询工具
- **Healthcare Bluebook** (healthcarebluebook.com) - 查询合理价格
- **FAIR Health Consumer** (fairhealthconsumer.org) - 医疗费用估算
- **New Choice Health** (newchoicehealth.com) - 比价工具

### 代码查询
- **ICD-10 Code Lookup** (icd10data.com) - 诊断代码查询
- **CPT Code Lookup** (aapc.com) - 医疗程序代码查询

### 维权资源
- **Dollar For** (dollarfor.org) - 免费账单谈判
- **Patient Advocate Foundation** (patientadvocate.org)
- **Consumer Financial Protection Bureau** (cfpb.gov)

---

## 检查完成后

✅ **如果没有发现错误**:
- 可以开始谈判降低金额
- 申请慈善医疗减免
- 询问付款计划

❌ **如果发现错误**:
- 立即提出争议
- 不要支付有争议的金额
- 要求更正后的账单

---

**记住**: 80%的医疗账单都有错误。仔细检查是你的权利,也是保护自己的第一步!
