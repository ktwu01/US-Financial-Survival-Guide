## 什么是"住房优先"原则

### 定义

**Housing First** 是一个在美国无家可归者服务领域广泛采用的政策理念:

> "住房是所有其他生活稳定因素的前提条件。没有稳定住所,无法解决就业、健康、教育等任何其他问题。"

### 起源

- **1992年**: 美国心理学家Sam Tsemberis在纽约创立Pathways to Housing,首次提出Housing First模型
- **核心发现**: 传统模式要求无家可归者"先戒毒/找工作,再给住房"效果很差;反转顺序后,成功率显著提高
- **科学证据**: 多个随机对照试验证明Housing First模型在稳定性、健康、成本效益上均优于传统模式

### 核心逻辑

```
稳定住房 → 稳定地址 → 稳定睡眠 → 稳定情绪 → 稳定工作 → 稳定收入 → 摆脱贫困
```

**失去住房后的反向螺旋** (见[失去住房的连锁反应](../crisis/losing-housing-cascade.md)):

```
无家可归 → 无法工作 → 无收入 → 健康恶化 → 社会孤立 → 永久性贫困
```

## 为什么住房是不可妥协的

### 1. 住房是所有其他功能的前提

**没有住房,你无法**:

| 需求 | 为什么需要住房 |
|------|---------------|
| **找工作** | 需要地址填写简历;需要稳定网络投递申请;需要干净衣服和良好精神面貌参加面试 |
| **保住现有工作** | 需要稳定睡眠保证出勤;需要洗澡保持个人卫生;需要充电设备(手机/电脑)维持沟通 |
| **维持健康** | 需要稳定睡眠和营养;需要地址接收医疗账单和保险信;需要储存药物和医疗用品 |
| **保持社交关系** | 需要私密空间进行视频通话;需要地址接收重要文件;无家可归带来巨大社会羞耻感 |
| **获得政府福利** | 几乎所有福利申请都需要固定地址;无家可归者难以接收政府通知和支票 |
| **使用银行服务** | 银行需要地址开户和寄送月结单;无地址可能导致账户被关闭 |

### 2. 住房损失是不可逆的临界点

**其他债务不还的后果** (可逆):
- 信用卡欠款 → 信用分数下降 → 但2-7年可恢复
- 医疗账单不付 → 进入催收 → 但不影响立即生存能力
- 学生贷款拖欠 → 工资被扣 → 但可申请hardship forbearance

**住房损失的后果** (不可逆):
- 失去住房 → 进入无家可归 → **极难恢复**
- 恢复成本极高: 需要first month + last month + security deposit (通常3-4个月租金)
- 无家可归记录影响未来租房申请
- 精神和身体健康快速恶化

### 3. 数据支持

**Housing First模型的科学证据**:

| 研究 | 发现 |
|------|------|
| **加拿大At Home/Chez Soi项目** (2009-2013) | Housing First参与者住房稳定率比对照组高61%;生活质量显著改善 |
| **美国HUD资助研究** | Housing First降低了急诊室使用率(-35%)和住院率(-33%),节省医疗成本 |
| **芬兰国家政策** | 采用Housing First后,芬兰长期无家可归人数从2008年的1,345人降至2022年的503人 |
| **Utah州案例** | 2005-2015年间,通过Housing First将长期无家可归人数减少91% |

**结论**: 即使对于有严重毒瘾、精神疾病的长期流浪者,**先提供住房**的模式也比"先康复再给住房"更有效。

## 财务优先级框架

### 三级优先级体系

#### 第一级：生存基本需求 (绝对优先)

**必须支付**:
1. **房租/抵押贷款** - 保住住房
2. **水电气** - 维持居住功能 (但有2-3个月缓冲期)
3. **食物** - 生存必需 (优先申请SNAP食品券)
4. **基本交通** - 上班/上学所需 (公交卡/最低汽车保险)

**为什么这些优先**:
- 失去后果立即且严重
- 恢复成本极高
- 影响所有其他生活功能

#### 第二级：法律和身份维持

**重要但有缓冲期**:
5. **学费** (对F-1学生) - 维持身份 (但可申请延期/减免)
6. **医疗保险** - 避免未来灾难性医疗费用 (F-1强制要求)
7. **汽车保险** (如必需) - 法律要求
8. **税款** - IRS可协商分期付款

**为什么次优先**:
- 有一定缓冲期(几周到几个月)
- 可以协商延期或分期
- 未立即影响住房稳定性

#### 第三级：可协商债务

**可以暂时停止支付**:
9. **信用卡最低还款** - 只影响信用分数
10. **医疗账单** - 可无限期协商
11. **学生贷款** (联邦) - 可申请forbearance或income-driven repayment
12. **个人贷款** - 可协商延期
13. **其他消费债务** - 最低优先级

**为什么最后**:
- 不会立即影响生存
- 有多种协商选项
- 信用分数可以在未来2-7年内恢复

## 常见债务优先级排序

### 情景1：只够支付部分账单

**你有**: $1,500
**账单**:
- 房租: $1,200
- 信用卡最低还款: $200
- 医疗账单: $500
- 水电费: $150

**正确做法**:
1. ✅ 支付房租全额 ($1,200)
2. ✅ 支付水电费 ($150) - 总计 $1,350
3. ❌ 信用卡不付
4. ❌ 医疗账单不付

**剩余$150**: 买食物

**联系债权人**:
- 信用卡: 打电话说明情况,要求skip a payment或hardship program
- 医疗账单: 申请financial assistance或payment plan

---

### 情景2：信用卡 vs 房租

**错误想法**: "信用卡APR 25%,利息太高了,我先还信用卡再说"

**正确逻辑**:

| 对比维度 | 信用卡欠款 | 房租拖欠 |
|---------|-----------|---------|
| **后果时间线** | 30天后影响信用分数 | 3-5天收到eviction notice |
| **法律后果** | 无(民事债务) | 驱逐诉讼(court record) |
| **立即影响** | 无 | 失去住房 → 失去一切 |
| **恢复难度** | 2-7年可修复 | 极难恢复(需3-4个月租金) |
| **协商空间** | 可申请hardship program | 房东通常不接受延期 |

**结论**: 即使信用卡利息再高,**也要先保住住房**。信用分数以后可以恢复,但无家可归很难翻身。

---

### 情景3：医疗账单 vs 房租

**正确答案**: **绝对优先支付房租**

**医疗账单的特点**:
- ✅ 不会影响住房
- ✅ 可无限期协商(医院通常接受$25/月的payment plan)
- ✅ 可申请charity care追溯减免
- ✅ 7年后从信用报告消失
- ✅ 医疗债务对信用分数影响比其他债务小

**房租的特点**:
- ❌ 拖欠7-14天后收到pay or quit notice
- ❌ 拖欠30天可能被驱逐
- ❌ 驱逐记录影响未来5-7年租房
- ❌ 失去住房导致连锁崩溃

**真实案例**:
> Sarah收到$15,000医疗账单,她用信用卡支付了,想"保护信用分数"。结果信用卡高额还款让她付不起房租,被驱逐。
>
> **正确做法**: 不付医疗账单,联系医院申请charity care → 可能获得80-100%减免

---

### 情景4：学生贷款 vs 房租 (F-1学生不适用,但对毕业后H-1B/绿卡持有者重要)

**联邦学生贷款特点**:
- ✅ 可申请income-driven repayment (IDR) - 月还款可低至$0
- ✅ 可申请forbearance或deferment暂停还款6-12个月
- ✅ 拖欠后有9个月缓冲期才default
- ✅ 即使default,可通过loan rehabilitation恢复

**优先级**: **房租 >> 学生贷款**

**行动步骤**:
1. 保住房租
2. 立即联系loan servicer申请hardship forbearance
3. 申请IDR计划(月供可低至收入的10%)

## F-1学生的特殊考量

### 为什么F-1学生住房更加关键

对于F-1学生,住房不仅是生存问题,还是**法律身份问题**:

| 维度 | 影响 |
|------|------|
| **SEVIS地址报告** | 必须在搬家后10天内更新SEVIS地址;无固定地址 = 无法合规报告 |
| **学校通知** | I-20更新、USCIS通知、OPT/CPT批准信都寄送到登记地址 |
| **就业要求** | I-9表格必须填写地址;雇主需要W-4地址;无地址 = 无法合法工作 |
| **银行和信用** | 银行账户需要地址;信用卡需要billing address;无地址可能被关户 |
| **签证维持** | DSO可能因"未维持身份"而terminate SEVIS |

### F-1学生的住房优先级调整

**标准优先级** (美国公民/绿卡):
1. 房租
2. 食物
3. 医疗保险
4. 学费(可贷款)

**F-1学生优先级**:
1. **房租** - 维持地址 = 维持身份
2. **学费** - 不full-time enrollment = 失去F-1身份
3. **医疗保险** - 学校强制要求
4. 食物(可申请食品银行)
5. 其他一切

**关键差异**: F-1学生的学费优先级高于一般美国学生,因为关系到身份维持。

---

### F-1学生的额外风险

**情景**: 你失去住房,暂时住朋友家/车里

**身份风险**:
1. **SEVIS地址不合规** - 如果你填朋友地址,但实际不住那里,属于虚假信息
2. **DSO面谈风险** - 如果DSO发现你无固定住所,可能terminate你的SEVIS
3. **I-20续签问题** - 无稳定地址可能被视为"unable to maintain status"
4. **签证续签** - 领事官可能质疑你的财务能力

**真实案例**:
> Kevin (F-1学生) 因拖欠房租被驱逐,暂时住车里。他没有及时更新SEVIS地址,DSO发现后terminate了他的SEVIS。尽管他后来找到新住房,但已失去身份,被要求离境。

**教训**: 对F-1学生,住房 = 身份,任何风险都不可接受。

## 何时可以不付房租

### 唯一可以不付房租的情况

**只有这种情况可以考虑不付房租**:

1. ✅ 你已经找到了新的住房(已签lease,已付deposit)
2. ✅ 新住房价格更便宜,长期可负担
3. ✅ 搬家成本已计算在内
4. ✅ 当前房东已多次违法(严重维修问题、骚扰、非法涨租等)
5. ✅ 你已咨询过tenant rights组织或律师

**即使满足以上条件,也要**:
- 发送formal notice给房东说明不付款原因
- 保存所有证据(照片、邮件、维修请求记录)
- 咨询法律援助组织
- 做好应对可能的驱逐诉讼的准备

### 绝对不要停止付房租的情况

❌ **"反正我要搬走了,最后一个月不付了"**
- 后果: 驱逐记录 + 欠款记录 + 影响未来租房
- 正确做法: 提前30-60天notice,按时付完最后一个月

❌ **"房东不修东西,我扣房租来修"**
- 在某些州这是非法的(即使房东违约)
- 正确做法: Repair and deduct或rent escrow(具体看州法)

❌ **"我要把房租拿去付信用卡/医疗账单"**
- 后果: 失去住房 → 失去一切
- 正确做法: 先付房租,再协商其他债务

❌ **"我找到工作了,先垫付deposit,下个月再补房租"**
- 风险: 如果新工作没到位/被裁,你已经无家可归
- 正确做法: 协商房租延期,或借钱付房租

## 紧急住房基金规划

### 为什么需要住房专用基金

**标准建议**: "建立3-6个月emergency fund"

**现实**: F-1学生很难存够3-6个月生活费

**Housing First策略**: **优先建立2-3个月的房租专用基金**

**逻辑**:
- 如果你只能存$3,000,与其分散到"全面emergency fund",不如专门cover房租
- 房租是最不可妥协的开支
- 其他开支(食物、医疗)有更多应急资源

---

### 住房基金目标

**最低目标**: 2个月房租

**示例**:
- 月租金$1,000 → 存$2,000
- 月租金$1,500 → 存$3,000

**理由**:
- 大多数州驱逐程序需要30-60天
- 2个月可以给你找新工作/新住房的缓冲期

**理想目标**: 3个月房租 + 搬家押金

**示例**:
- 月租金$1,000 → 存$6,000 (3个月租金 + first/last month + deposit)

**理由**:
- 如果需要搬家,通常需要first month + last month + security deposit = 3个月租金
- 这笔钱可以让你在任何情况下保住住房

---

### 如何建立住房基金

#### 策略1: 单独账户隔离

**做法**:
1. 开一个独立储蓄账户(可以是同一银行)
2. 命名为"住房紧急基金" (心理提示作用)
3. 只有真正的住房危机才能动用

**心理学原理**: 钱混在一起容易被"借用";"专款专用"账户提高心理屏障

#### 策略2: 自动存款

**做法**:
- 每次收到工资后,自动转账$100-200到住房基金
- 不要手动操作,减少"这次就算了"的念头

#### 策略3: 意外收入全存

**做法**:
- 税务退款(tax refund)
- 奖学金/RA/TA多出来的钱
- 礼物、红包
- 任何"意外之财" → 100%存入住房基金

**目标**: 在1-2年内存够3个月房租

---

### 何时可以动用住房基金

**可以动用**:
✅ 失业超过1个月,房租无法支付
✅ 突发医疗导致无法工作
✅ 需要紧急搬家(房东驱逐、房屋不安全)

**不可以动用**:
❌ "我看中了一台新电脑,先用一下,下个月补回来" - ❌
❌ "朋友结婚,我要买机票回国" - ❌
❌ "车坏了,修车费$1,000" - ❌ (除非车是上班必需)

**原则**: 只有"失去住房"的风险才能动用,其他一切都可以协商/延期。

## 实战案例

### 案例1: Lisa的Housing First决策

**背景**:
- F-1学生,计算机专业,OPT期间
- 突然被layoff
- 月租金$1,200,存款$3,000
- 欠信用卡$5,000 (APR 24%)

**她的困境**:
- 信用卡每月最低还款$150
- 信用卡利息每月$100
- 房租到期还有5天

**错误做法A**: "信用卡APR 24%太高了,我先还$1,000信用卡,降低债务"
- 结果: 剩$2,000,只够1.5个月房租
- 3周后付不起房租,被驱逐
- 失去住房 → 无法面试 → 无法找到工作

**错误做法B**: "我平均分配,$1,500房租,$1,500信用卡"
- 结果: 房东拒绝接受部分付款,发出pay or quit notice
- Lisa被迫刷爆另一张信用卡才凑够房租
- 陷入更深的债务螺旋

**正确做法 (Housing First)**:
1. ✅ 用$2,400支付2个月房租(提前给房东,说明情况)
2. ✅ 用$300买食物和公交卡(维持基本生存和求职)
3. ✅ 剩$300作为小额应急
4. ✅ 信用卡完全不还,立即打电话申请hardship program
5. ✅ 联系信用卡公司说明失业情况,要求降低APR或skip payment

**结果**:
- 保住住房2个月
- 在第6周找到新工作
- 3个月后开始还信用卡
- 信用分数虽然受损(从720降到650),但2年后恢复到700+
- **最重要**: 没有失去住房,保住了稳定性和F-1身份

**对比**:
- 如果做法A或B → 失去住房 → 至少6-12个月才能恢复 → 可能失去F-1身份 → 被迫离境

**教训**: **信用分数可以恢复,但失去住房几乎不可逆。**

---

### 案例2: Michael的医疗账单危机

**背景**:
- F-1学生,化学PhD,RA月收入$1,800
- 月租金$900
- 突发阑尾炎手术,收到医疗账单$28,000
- 有学校保险,但保险只cover $15,000,自付$13,000

**他的错误决策**:
1. 恐慌之下,用信用卡支付了$13,000医疗账单(想"避免催收")
2. 信用卡账单来了,最低还款$400/月
3. $1,800收入 - $900房租 - $400信用卡 = $500 (食物+交通+其他)
4. 非常紧张,开始用另一张信用卡支付日常开销
5. 3个月后,两张信用卡都maxed out,开始付不起房租

**结果**:
- 第4个月收到eviction notice
- 被迫向父母借钱$3,000才保住住房
- 陷入长期债务螺旋

**正确做法 (Housing First)**:
1. ✅ 收到$13,000账单后,**完全不付**
2. ✅ 继续正常支付$900房租
3. ✅ 立即联系医院billing department,申请financial assistance
4. ✅ 提交收入证明(RA stipend $1,800/月 = 年收入$21,600,远低于很多医院的300% FPL标准)
5. ✅ 获得80% charity care减免,自付降为$2,600
6. ✅ 与医院协商payment plan: $50/月 × 52个月

**最终成本对比**:

| 方案 | Michael的错误做法 | 正确做法 (Housing First) |
|------|------------------|------------------------|
| **医疗账单** | $13,000 (用信用卡付) | $2,600 (charity care减免后) |
| **利息成本** | $8,500 (24% APR × 5年) | $0 (医院payment plan无息) |
| **住房风险** | 差点被驱逐 | 稳定 |
| **信用影响** | 信用卡maxed out,利用率90%,信用分数暴跌 | 无影响(charity care不影响信用) |
| **总成本** | $21,500 + 巨大压力 | $2,600 + 零压力 |

**教训**:
1. **绝对不要用信用卡支付医疗账单**
2. **先保住住房,再处理医疗债务**
3. **医疗债务是最"温和"的债务,有无限协商空间**

### 案例3: Zhang的学费vs房租困境

**背景**:
- F-1学生,本科大三,学费$25,000/年
- 父母原本支付学费,但突然失业,无法再汇款
- Zhang需要自己cover春季学期学费$12,500
- 月租金$800,有室友
- 打工收入$1,200/月(校内工作)
- 存款$5,000

**他的困境**:
- 学校deadline: 1月15日必须付学费,否则drop所有课程
- 房租deadline: 1月1日
- drop课程 = 失去full-time status = 失去F-1身份

**他的错误思路**:
"学费关系到身份,更重要,我先付学费,房租欠2个月,等我找到校外工作再补"

**为什么这是错的**:
1. F-1学生不能校外工作(除非CPT/OPT)
2. 欠2个月房租($1,600) → 100%会被驱逐
3. 失去住房 → 无法上课 → 同样失去full-time status
4. 被驱逐的记录影响未来租房 → 可能找不到新住房 → 同样失去身份

**正确做法 (分步处理)**:

**第1步: 保住住房**(优先级1)
- 用$2,400支付3个月房租(1月-3月)
- 剩余存款: $2,600

**第2步: 争取学费延期**(优先级2)
- 立即联系学校financial aid office和international student office
- 说明情况: 父母突然失业,已付房租保证生存,需要学费延期
- 申请emergency loan或payment plan

**可能结果**:
1. **最好情况**: 学校提供emergency loan或允许分期付款($1,000/月 × 12个月)
2. **次好情况**: 学校允许late payment,给1-2个月grace period
3. **最坏情况**: 学校不允许延期,must pay by deadline

**第3步: 如果学校完全不给延期** (极端情况)

这时才是真正的两难:

**选项A**: 付学费,不付房租
- 结果: 保住F-1身份,但1个月后被驱逐 → 无家可归 → 无法上课 → 最终还是失去身份

**选项B**: 付房租,不付学费
- 结果: 被drop课程,失去F-1身份,但有稳定住房3个月来处理后续

**哪个更好? → 选项B**

**理由**:
1. 有住房的情况下,可以申请readmission/reinstatement
2. 可以转学到学费更低的学校(community college)
3. 可以回国重新申请F-1签证
4. 如果无家可归,上述所有选项都不可行

**实际结果** (真实案例):
- Zhang选择了正确做法,先付房租
- 学校financial aid office给了他emergency loan $10,000 (无息,毕业后1年开始还)
- 他用loan付了学费,保住了住房和身份
- 大四毕业后找到工作,2年内还清了loan

**教训**:
1. **即使对F-1学生,住房仍然是第一优先级**
2. **学校通常比你想象的更flexible,一定要先沟通**
3. **如果真的两难,有住房的情况下处理身份问题远比无家可归容易**

## 常见误区

### 误区1: "我要保护我的信用分数"

**错误想法**:
"我的信用分数720,如果不还信用卡会降到600,我花了3年才建立起来的,不能毁掉"

**真相**:
- 信用分数可以在2-7年内恢复
- 失去住房的后果比信用分数下降严重100倍
- 很多F-1学生毕业后回国,美国信用分数对他们长期意义有限

**正确思维**:
"信用分数是工具,不是目的。它的作用是帮助我租房、贷款、找工作。如果为了保护信用分数而失去住房,本末倒置。"

**数据**:
- 信用分数从720降到600: 影响2-5年,可逐步恢复
- 驱逐记录: 影响5-7年,极难消除,基本无法租房

---

### 误区2: "医疗账单不付会被起诉"

**错误想法**:
"医疗账单是合法债务,如果不付,医院会起诉我,我会有court record,影响身份"

**真相**:
- 大部分医院不会为小额账单起诉(成本太高)
- 即使起诉,medical debt判决不影响F-1身份(民事判决,非刑事)
- 医院更愿意协商payment plan或charity care

**统计数据**:
- 美国约2300万人有medical debt in collections
- 医院起诉的比例 < 5%
- 绝大多数案例是协商分期付款

---

### 误区3: "房东一定会驱逐我"

**错误想法**:
"如果我晚付房租一周,房东会立即驱逐我"

**真相**:
- 驱逐程序通常需要30-60天
- 大多数房东更愿意接受晚付,而不是驱逐(空置成本更高)
- 主动沟通可以避免很多问题

**正确做法**:
1. 在rent due date之前联系房东
2. 说明情况(失业、医疗紧急)
3. 提出具体还款计划(例如: "我可以在15号付$600,月底付剩下的$600")
4. 书面确认(email)

**很多房东会接受的原因**:
- 驱逐流程成本$3,000-5,000 (律师费+法庭费用+lost rent)
- 重新找租客需要1-2个月
- 好租客(按时付款记录)值得给一次机会

---

### 误区4: "我应该借高利贷付房租"

**错误想法**:
"付不起房租,我去payday loan借$1,000,APR虽然300%,但至少保住了住房"

**真相**:
- Payday loan是债务陷阱,几乎不可能还清
- 2周后你不仅要付房租,还要还payday loan + 300% APR
- 这会让你更快进入债务螺旋

**正确做法**:
1. 申请emergency assistance (211, 教会, 非营利组织)
2. 向朋友/家人借款(无息或低息)
3. 联系房东协商延期
4. 申请学校emergency fund (很多大学有)
5. 考虑找室友或搬到更便宜的地方

**最后手段**: 信用卡cash advance (虽然APR 25-30%,但仍比payday loan的300-400%好)

---

### 误区5: "我会很快找到工作,可以先欠房租"

**错误想法**:
"我简历很好,肯定能在2周内找到工作,我先把房租的钱用来买面试的衣服和交通"

**真相**:
- 平均求职周期: 3-6个月 (即使对qualified候选人)
- 即使拿到offer,通常需要2-4周才能拿到第一份paycheck
- 失业的心理压力会降低面试表现
- 失去住房会让求职几乎不可能

**正确做法**:
1. 保住住房是找工作的前提
2. 用节俭策略降低其他开支(食物银行, 公共交通)
3. 申请失业救济(F-1学生可能不eligible,但H-1B/绿卡可以)
4. 接受任何临时工作(gig economy, tutoring)

## 决策树

### 决策树1: 付不起房租时怎么办

```
付不起房租
    ↓
❓ 距离due date还有多久?
    ├─ >14天 → ✅ 有时间协商
    │   ↓
    │   1. 计算差额: 差多少钱?
    │   2. 削减所有非必要开支
    │   3. 寻找临时收入(卖物品,gig work)
    │   4. 联系房东提前说明,请求延期
    │   └─ 房东同意?
    │       ├─ 是 → ✅ 书面确认,按约定付款
    │       └─ 否 → 继续下面流程
    │
    └─ <14天 → ⚠️ 紧急情况
        ↓
        ❓ 有任何存款/资产吗?
        ├─ 有 → ✅ 立即变现
        │   - 卖非必需品(电子产品,家具)
        │   - 向朋友/家人借款
        │   - 信用卡cash advance(最后手段)
        │   └─ 凑够了?
        │       ├─ 是 → ✅ 立即付款
        │       └─ 否 → 继续下面
        │
        └─ 无 → ⚠️ 求助资源
            ↓
            1. ✅ 拨打211查询emergency rent assistance
            2. ✅ 联系当地非营利组织(教会, Salvation Army, Catholic Charities)
            3. ✅ 联系学校international student office的emergency fund
            4. ✅ 申请GoFundMe等众筹(如果愿意)
            └─ 仍不够?
                ↓
                ❓ 有更便宜的住房选项吗?
                ├─ 是 → ✅ 立即搬家
                │   - 找室友(降低房租)
                │   - 搬到更便宜的区域
                │   - 短期住朋友家(过渡期)
                │
                └─ 否 → ⚠️ 最后方案
                    ↓
                    1. 联系房东坦诚沟通
                    2. 提出分期付款(例如: 每周$300)
                    3. 提供离开计划(我会在X月搬走)

```

## 总结: Housing First的核心原则

### 为什么住房是第一优先级

1. **住房是所有其他生活功能的前提**
   - 没有住房,无法工作、上学、求职、维持健康

2. **住房损失是不可逆的临界点**
   - 其他债务(信用卡、医疗)可以协商、延期、减免
   - 失去住房后极难恢复(需3-4个月租金)

3. **对F-1学生,住房=身份**
   - SEVIS地址报告要求
   - 学校和USCIS通信需要地址
   - 无家可归可能导致身份terminate

4. **科学证据支持**
   - Housing First模型在全球被证明最有效
   - 即使对严重毒瘾/精神疾病患者,"先给住房"也比"先康复再给住房"成功率高

```
第一级: 生存三要素
- 房租 (住房)
- 水电气 (居住功能)
- 食物 (生存必需)

第二级: 身份和法律
- 学费 (F-1学生)
- 医疗保险 (F-1强制)
- 法律要求 (汽车保险)

第三级: 可协商债务
- 信用卡 (可hardship program)
- 医疗账单 (可charity care)
- 学生贷款 (可forbearance)
- 其他一切 (都可以晚点处理)
```
