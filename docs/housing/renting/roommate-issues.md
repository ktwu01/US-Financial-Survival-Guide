# 室友纠纷处理

## 预防胜于治疗

### 搬入前做什么

**面试潜在室友**:
- 生活习惯（作息时间）
- 清洁标准
- 访客政策
- 噪音容忍度
- 财务责任
- 宠物
- 共享物品

**书面协议(Roommate Agreement)**:
- 即使是朋友也要写
- 房租分配
- 水电费分摊
- 清洁责任
- 共用空间规则
- 访客规则
- 提前搬走条款

## 常见室友问题

### 问题1: 不付账单/房租

**症状**:
- 总是"忘记"转账
- 拖延付款
- 找借口

**解决**:
1. **书面提醒**:
   ```
   Hi [名字], just a reminder that your share of
   [rent/utilities] for this month is $[金额].
   Please Venmo me by [日期]. Thanks!
   ```

2. **设置deadline**: 明确日期

3. **记录**: 保存所有通信

4. **如果继续**:
   - 通知房东（如果你们联名租约）
   - 考虑legal action
   - Small claims court

**预防**:
- 设置自动提醒
- 使用Splitwise app
- 月初就要求付款

### 问题2: 不清洁

**症状**:
- 不洗碗
- 厨房/浴室脏乱
- 垃圾不扔
- 公共区域乱

**解决**:
1. **清洁责任表**:
   - 轮流制度
   - 具体任务
   - 时间表

2. **友好提醒**:
   ```
   Hey, I noticed the dishes have been sitting for
   a few days. Could you take care of them today?
   ```

3. **House Meeting**:
   - 讨论标准
   - 设定规则
   - 定期检查

4. **最后手段**:
   - 雇清洁工，分摊费用
   - 考虑换室友

### 问题3: 噪音问题

**症状**:
- 深夜大声
- 频繁party
- 音乐太大声

**解决**:
1. **沟通作息时间**:
   - 什么时间需要安静
   - quiet hours (如10pm-8am)

2. **当场提醒**:
   - 礼貌但明确
   - "Could you keep it down? I have class/work tomorrow"

3. **书面规则**:
   - 加入roommate agreement
   - 设定合理标准

4. **升级**:
   - 通知RA（宿舍）
   - 通知房东
   - 报警（极端情况）

### 问题4: 未经允许使用物品

**症状**:
- 吃你的食物
- 用你的物品
- 未经许可进你房间

**解决**:
1. **明确界限**:
   ```
   Hey, I noticed my [物品] was used.
   Please ask me first next time. Thanks!
   ```

2. **标记物品**:
   - 食物写名字
   - 贵重物品锁起来

3. **严重情况**:
   - 房间安装锁
   - 报告房东
   - 可能theft report

### 问题5: 访客问题

**症状**:
- 频繁过夜客人
- 实际上的"室友"
- 陌生人

**解决**:
1. **访客规则**:
   - 过夜需要提前告知
   - 限制过夜频率
   - 长期客人需同意

2. **沟通**:
   ```
   I notice [客人] has been staying over a lot.
   Can we discuss visitor guidelines?
   ```

3. **房东介入**:
   - 如果违反租约
   - 未经授权的occupant

## 室友冲突升级阶段

### Level 1: 直接沟通
- 私下谈话
- 保持冷静
- 具体问题具体说
- 寻求解决方案

### Level 2: 书面沟通
- Email/text记录
- 明确期望
- 设定deadline
- 保留证据

### Level 3: 中介
- RA（宿舍）
- 房东
- 第三方室友

### Level 4: 法律/搬走
- 咨询律师
- 解除租约
- 寻找新住处

## 书面沟通模板

### 友好提醒
```
Hey [名字],

I wanted to talk about [问题]. I've noticed [具体情况].

Could we [解决方案]? I think this would help us both.

Let me know what you think!

Thanks,
[你的名字]
```

### 正式警告
```
Hi [名字],

I need to address [问题] which has been ongoing.

Despite previous conversations, [具体情况] continues.

I request that [具体要求] by [日期].

If this continues, I will need to [后续步骤].

[你的名字]
```

## 何时离开

### 危险信号
- 人身威胁
- 盗窃
- 暴力
- 非法活动
- 严重影响学业/工作
- 精神健康受损

### 提前终止租约

**查看租约**:
- 提前终止条款
- 罚金
- 找替代者

**与房东协商**:
- 说明情况
- 提供证据
- 寻求理解
- 提出替代方案

**法律原因**:
- 家庭暴力受害者（各州有保护）
- 不宜居住条件
- 房东违约

## 联名租约风险

### 什么是Joint Lease
- 所有室友共同签约
- 所有人共同负责全部房租
- 一人不付，其他人要补

### 保护自己
- 只和信任的人联名
- 保留付款记录
- 如果一人不付，你付全租后可以sue他们

### 替代方案
- Individual leases
- 或master tenant + sublet

## 财务纠纷

### 水电费分摊
- 平均分（最简单）
- 按使用（有争议）
- 按房间大小
- 使用Splitwise app

### 共同购买物品
- 保留收据
- 记录谁买了什么
- 搬走时如何分

### 损坏责任
- 谁损坏谁负责
- 拍照证明
- 扣押金时可能影响所有人

## 预防最佳实践

### ✅ Do
- 开始就设定规则
- 定期house meetings
- 保持清晰沟通
- 尊重个人空间
- 及时解决小问题
- 保持财务透明

### ❌ Don't
- 被动攻击行为
- 背后议论
- 小事拖成大事
- 假设对方知道你的期望
- 情绪化反应

⚠️ **记住**: 大多数室友问题可以通过良好沟通解决，但要有界限！
