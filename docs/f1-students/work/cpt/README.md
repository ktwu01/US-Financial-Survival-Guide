# CPT 工作指南

> Curricular Practical Training - 在读期间的合法工作授权

## 什么是CPT？

CPT是F-1学生在读期间进行实习或工作的合法授权，必须是课程要求的一部分。

## CPT类型

### Part-time CPT
- 每周工作不超过20小时
- 可以在上课期间使用
- 不影响OPT

### Full-time CPT
- 每周工作超过20小时
- 通常在暑假使用
- ⚠️ 使用12个月或以上将失去OPT资格

## 申请要求

- 已完成一学年全职学习（9个月）
- 工作必须与专业直接相关
- 需要有课程学分支持
- 雇主offer letter

## 关键注意事项

⚠️ **12个月Full-time CPT规则**: 使用满12个月（累计365天）将完全失去OPT资格

⚠️ **必须在工作开始前获批**: 未获批工作属于非法工作

⚠️ **工作必须在I-20授权期间**: 超期工作视为违反身份
