# OPT申请完整流程

## 申请时间表

### 时间窗口
- **最早**: 毕业前90天
- **最晚**: 毕业后60天
- **建议**: 毕业前60-75天申请

## 申请步骤

### 1. 向学校DSO申请I-20
- 填写OPT推荐表
- 获得DSO签字的新I-20
- 时间: 1-2周

### 2. 准备申请材料
- [ ] I-765表格（在线填写）
- [ ] 两张护照照片
- [ ] I-20复印件（所有页）
- [ ] I-94打印件
- [ ] 护照复印件（照片页和签证页）
- [ ] 以往EAD卡复印件（如有）
- [ ] $410申请费支票或汇票

### 3. 递交申请
- 邮寄到USCIS指定地址
- 建议使用USPS Priority Mail with tracking
- 保留邮寄凭证

### 4. 等待审理
- 正常审理时间: 3-5个月
- 可以在线查询状态
- 收到Receipt Notice后可用于就业证明

## 常见问题

**Q: 没有及时申请怎么办？**
A: 毕业后60天是硬性截止日期，过期无法申请

**Q: 可以加急处理吗？**
A: OPT不支持加急服务(Premium Processing)

**Q: 申请期间可以工作吗？**
A: 不可以，必须等EAD卡到手
