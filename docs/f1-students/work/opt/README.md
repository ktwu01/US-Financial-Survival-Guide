# OPT 工作指南

> Optional Practical Training (OPT) 是F-1学生毕业后合法工作的重要途径

## 什么是OPT？

OPT允许F-1学生在毕业前后在美国工作12个月（STEM专业可延长24个月）

## 关键时间节点

- 毕业前90天可申请
- 毕业后60天内必须递交申请
- 获批后有90天失业宽限期
- STEM延期需在首次OPT到期前90天申请

## 重要提醒

⚠️ **失业天数限制**: 累计90天失业将失去OPT资格
⚠️ **工作必须与专业相关**: 需要保留证明材料
⚠️ **及时向学校报告**: 雇主变更、地址变更必须10天内报告
